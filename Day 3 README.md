Project Overview

This project analyzes the relationship between students' lifestyle habits—such as screen time, food habits, 
and extracurricular activities—and their academic performance and health indicators.

The dataset includes the following key attributes:
 ID
Age
Gender
Height
Weight
Daily_Fruit_Serving
Daily_veggie_Serving
Dily sugar drink
Region
Screen Time 
Health Status
Region
BMI 

A dedicated Dashboard sheet was created including:

Key charts (scatter plot and bar chart)
Clean and structured layout for presentation
🔹 Features:
Slicers for:
Gender
Health Status
Dynamic filtering connected to all charts

A dropdown list was implemented for the Region column:

Ensures consistent data entry
Prevents manual input errors
Improves data quality

Poor lifestyle habits (high screen time, unhealthy food) impact both health and performance
